Equilibrium anchoring stabilizes multi-step rollout by separating steady-state anchoring from transient correction.

Abstract

We propose an equilibrium-anchored residual dynamics framework for stable multi-step prediction of soft and continuum robotic systems under actuation shift. The method learns an input-conditioned equilibrium reference from static data and a residual correction from dynamic trajectories, thereby separating steady-state anchoring from transient dynamics. We evaluate the framework on a tendon-driven 3D continuum arm, classical nonlinear benchmarks, and a soft-tail hardware testbed under actuation-frequency shift.

Method

Equilibrium Anchor

A static dataset is used to learn an actuation-conditioned equilibrium map, which provides a steady-state reference during rollout.

Residual Correction

A residual model is trained on dynamic trajectories to capture transient mismatch around the anchored state.

This explicit separation helps reduce drift in long recursive predictions while keeping the learned correction lightweight and control-friendly.

Results

Continuum Arm

The proposed method improves long-horizon prediction under out-of-distribution actuation and reduces backbone and tip prediction error relative to strong baselines.

Soft-Tail Hardware

On a custom soft-tail testbed under actuation-frequency shift, the method remains bounded and tracks oscillations more closely than anchor-only and residual-only variants.
