Stabilizing 3D Continuum-Arm Rollouts via Equilibrium Anchoring and Feature-Lifted Residual Learning

A hybrid predictor that learns an actuation-conditioned equilibrium reference from static data and a lightweight residual correction from dynamic trajectories.

Abstract

Multi-step motion prediction for continuum robots is difficult under actuation distribution shift because recursive errors can distort steady response and destabilize rollouts. We introduce an equilibrium-anchored residual-learning framework for a tendon-driven 3D continuum arm. The method learns a static equilibrium prior from inexpensive equilibrium data, uses it as a contractive rollout anchor, and fits a feature-lifted residual model for transient dynamics.

Method

The predictor separates steady-state anchoring from transient correction. At each rollout step, the current prediction is pulled toward an input-conditioned equilibrium estimate, then corrected by a residual model trained on anchored states.

Equilibrium anchor

Fits `P(u)` from the static `D2` dataset and uses a contractive update to suppress steady-state drift.

Residual correction

Fits a linear-in-parameters residual on `D1` transitions to recover transient behavior around the anchor.

Results

The hybrid model improves recursive rollout accuracy under stronger and faster actuation than used in training, while retaining stable behavior across the tested continuum-arm trajectories.
