Equilibrium anchoring stabilizes multi-step rollout by separating steady-state anchoring from transient correction.

Abstract

We propose an equilibrium-anchored residual dynamics framework for stable multi-step prediction of soft and continuum robotic systems under actuation shift. The method learns an input-conditioned equilibrium reference from static data and a residual correction from dynamic trajectories, thereby separating steady-state anchoring from transient dynamics. We evaluate the framework on a tendon-driven 3D continuum arm, classical nonlinear benchmarks, and a soft-tail hardware testbed under actuation-frequency shift.

Method

The predictor separates the steady-state component of the response from the transient correction learned from dynamic trajectories.

Equilibrium Anchor

A static dataset is used to learn an actuation-conditioned equilibrium map that provides a physically plausible steady-state reference during rollout.

Residual Correction

A feature-lifted residual model is trained on dynamic trajectories to capture transient mismatch around the anchored state.

Results

26%

Backbone RMSE Reduction

The hybrid predictor reduces backbone position error relative to the equilibrium-only anchor on the continuum-arm benchmark.

Tip RMSE Reduction

The method improves tip prediction accuracy while preserving stable recursive rollouts under stronger and faster actuation.

Hardware Improvement

On the soft-tail hardware testbed, Hybrid lowers RMSE by at least 74% relative to the residual-only model under frequency shift.
